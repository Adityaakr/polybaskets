## The **bet-lane** program

Companion Token betting lane for Polybaskets.

This workspace contains:
- `bet-lane` binary crate
- `bet-lane-app` business logic
- `bet-lane-client` generated Rust client
- `program/client` generated Rust client for the basket program IDL

### Building

```bash
cargo build
```

### Testing

```bash
cargo test
```
